GitHub Desktop에서 새로운 브랜치를 만드는 방법은 다음과 같습니다:

1.GitHub Desktop을 열고, 작업하고 있는 저장소를 선택합니다.

2.상단 메뉴에서 Branch > New Branch를 선택합니다.

3.새로운 브랜치 이름을 입력하고, Create Branch를 클릭합니다. 이때, 브랜치 이름으로 feat 또는 develop을 사용할 수 있습니다.

4.만든 브랜치로 전환하려면, 브랜치 드롭다운 메뉴에서 새로운 브랜치를 선택합니다.

이렇게 하면 feat 브랜치와 develop 브랜치를 만들고 전환할 수 있습니다. 브랜치를 전환하면, 해당 브랜치에서 코드를 추가하거나 수정할 수 있습니다. 작업이 완료되면, 변경 사항을 커밋하고 푸시할 수 있습니다.
